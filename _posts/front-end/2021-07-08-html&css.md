## 1. HTML 문법

약자: Hyper Text Markup Language

Hyper Text: 링크

- 태그: 꼬리표, 이름표

- 속성 (Attribute): 공백을 넣지 않는다. 홀따옴표나 쌍따옴표 모두 사용가능

  글로벌속성과 특정태그에만 있는 속성이 있다.

- 태그의 중첩 (Nesting Tags)

- 빈 태그 (Empty tag): 시작태그만 존재하고 닫는태그가 존재하지 않는 것. relace tag 라고도 한다.

- 공백

- 주석
