자바에서 클래스를 선언하는 형식 (시험)

[접근지정자] [기타제어자] class 클래스이름 { }

modifiers = 접근지정자 + 기타제어자

자바에서 함수 선언하는 형식 (시험)

```
[접근지정자] [기타제어자] 리턴자료형 함수명([매개변수]) {
	[return 리턴값;] 
}
```

매개변수 == 인자 == 파라미터 == 실인자, 가인자

args (arguments)
