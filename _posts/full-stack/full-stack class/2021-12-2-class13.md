**모든 a 태그에 이벤트 등록**

```html
<script>
    var links = document.querySelectorAll(".topnav a");

    for (var i = 0; i < links.length; i++) {
        links[i].onclick = function() {
            for (var i = 0; i < links.length; i++) {
                links[i].classList.remove("active");
            }
            this.classList.add("active");
        }
    }
</script>
```



> 자바스크립트는 i 를 두번 선언할 수 있다.
