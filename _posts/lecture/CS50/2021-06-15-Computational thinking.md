## C언어

실습사이트

<https://sandbox.cs50.io/>{:target="_blank"}

GitHub 계정을 연동해 사용하므로 계정이 없다면 생성합니다.

`clang fileName.c` 컴파일 :  a.out 파일을 만듭니다.

`./a.out` 파일을 실행합니다. 

- `.` 현재 폴더를 나타냅니다.

`$` prompt

`ls` list

- `*` 가 붙은 것은 실행가능한 파일이며 없는 것은 소스코드이다.

`rm` remove

`mkdir` make directory

`rmdir` remove directory
